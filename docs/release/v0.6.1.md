# v0.6.1 Release Notes

## Summary

v0.6.1 is a desktop cache hotfix for v0.6.0. It keeps the automatic prompt dispatch feature intact and fixes a case where the Electron desktop window could continue showing an older cached frontend after the Docker stack had already pulled the v0.6.0 images.

## Fixed

- Clears the Electron workbench HTTP cache before opening the Web workspace from the desktop shell.
- Marks `index.html` and SPA fallback responses as non-cacheable in the Web Nginx config.
- Keeps hashed `/assets/*` files cacheable and immutable, so normal static asset performance is preserved.

## User Impact

- After upgrading to v0.6.1, the creation page should immediately show real-time prompt dispatch feedback such as `已解析到 2 个任务` for `{prompt:"a"},{prompt:"b"}`.
- Existing v0.6.0 data, conversations, tasks, users, and settings are preserved.
- No database migration is introduced by this release.
