# v0.6.0 Release Notes

## Summary

v0.6.0 adds automatic prompt dispatch to the creation and workspace composers. Users can paste json-like prompt objects such as `{prompt:"first"},{prompt:"second"}` and create multiple independent image tasks in one workspace session while sharing the currently selected model, assets, size, and batch count.

## Added

- Parses json-like prompt input with one `prompt` field per object.
- Shows real-time composer feedback such as `已解析到 2 个任务`.
- Creates parsed prompts as separate task records in the same conversation.
- Keeps recent tasks, history, and library views treating those tasks as independent items.
- Adds body-level image preview for workspace task images, using 100% image scale and click-to-close behavior.

## Changed

- Workspace and creation page submissions now fan out parsed prompt lists on the frontend while preserving the existing single-task backend API.
- Task and batch-result image previews now share a lightweight full-screen image lightbox.
